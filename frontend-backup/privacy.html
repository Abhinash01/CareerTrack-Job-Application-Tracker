Data Protection

Privacy Policy

This Privacy Policy explains how CareerTrack handles user data and application information.

1. Information We Collect

CareerTrack may collect basic user information such as name, email address, profile details, job application records, interview dates, notes and status updates.

2. How We Use Your Information

The information is used to help users manage job and internship applications, display dashboard analytics, track progress, and provide reminders or notifications.

3. Application Data

Users can store company names, job roles, locations, application status, applied dates, interview dates, source details and personal notes related to their career applications.

4. Local Storage Usage

The frontend version may use browser localStorage to temporarily store application data for testing. In the full-stack version, this data will be stored securely in a database.

5. Database Storage

In the backend version, user data and application records will be stored using MongoDB. Access to user-specific data will be protected through authentication and authorization.

6. Authentication

CareerTrack will use secure login methods such as hashed passwords and JWT-based authentication to protect user accounts and prevent unauthorized access.

7. Data Sharing

CareerTrack does not sell or share user information with third parties. User data is intended only for personal career tracking and platform functionality.

8. Email Notifications

If email notification features are enabled, CareerTrack may use the user's email address to send interview reminders, follow-up alerts or account-related messages.

9. User Control

Users may add, update or delete their application records. In the full-stack version, users will be able to manage profile information and account-related data.

10. Contact

For privacy-related questions, users can contact the CareerTrack support team through the Contact page.
